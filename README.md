# WebScraper
This is an example Maven project written in Java 8.
## Technology stack
* Jsoup 1.11.1
* HttpClient 4.5.3
* Commons CLI 1.4
* Log4j 4.5.3
* JUnit 4.12
* Mockito 2.9.0
* Maven 3.3.9
## Options
<pre>
usage: webscraper
 -d,--debug      debug mode
 -h,--help       print this message
 -p,--profile <arg>   input profile name: ceneo-list (default), ceneo-box
 -t,--type <arg>      output type: xml (default)
 -u,--url <arg>       source url (required)
</pre>
